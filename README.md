# Car Sales Performance Analysis Dashboard

An interactive Power BI dashboard developed to analyze vehicle sales performance and generate business insights for data-driven decision-making.

---

## 📌 Business Objective

This project aims to analyze sales performance across different vehicle brands, body styles, colors, dealers, and time periods. The dashboard helps stakeholders monitor key business metrics, identify sales trends, and support strategic decisions for inventory planning and sales optimization.

---

## 📂 Dataset

- **Source:** Car Sales Dataset
- **Records:** 5,000+ vehicle sales transactions
- **Key Features:**
  - Brand
  - Dealer
  - Body Style
  - Color
  - Engine
  - Transmission
  - Price
  - Date

---

## 🛠 Data Preparation

Data was cleaned and transformed using **Power Query** by:

- Handling missing values
- Removing duplicate records
- Correcting data types
- Creating calculated columns
- Preparing data model for analysis

---

## 📊 Dashboard Features

### Key Performance Indicators (KPIs)

- Total Sales
- Units Sold
- Average Selling Price

### Interactive Analysis

- Sales performance by brand
- Vehicle body style analysis
- Color distribution
- Weekly sales trend
- Dealer performance
- Engine & transmission analysis

### Interactive Functions

- Dynamic slicers
- Cross-filtering
- Drill-through analysis

## 🔍 Key Insights

- SUV was the highest-performing vehicle segment.
- Chevrolet generated the highest total sales.
- Pale White was the most popular vehicle color.
- Oldsmobile recorded the highest average selling price.
- Sales performance varied significantly across dealers and vehicle categories.

---

## 💡 Business Recommendations

- Increase inventory for high-demand vehicle segments.
- Prioritize marketing campaigns for top-performing brands.
- Optimize pricing strategies for premium vehicles.
- Improve inventory allocation based on customer purchasing trends.
